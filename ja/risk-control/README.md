1.デジタル資産に関連するリスク

-デジタル資産商品への投資の価格および価値、ならびにその収益（もしあれば）は変動する可能性があり、お客様の利益に反する方向に動く場合があります。個々のデジタル資産商品の価格は下落する可能性があり、場合によっては無価値になることもあります。デジタル資産商品の売買・投資に内在するリスクとして、利益を得るのではなく損失を被る可能性があります。

-デジタル資産は法定通貨ではなく、政府や金・銀などの商品による裏付けがありません。危機においてビットコインおよび／またはイーサリアムおよび／またはその他のデジタル資産の価値を保護するための是正措置を講じたり、追加のデジタル資産を発行したりできる中央銀行は存在しません。むしろ、ビットコイン、イーサリアムおよびその他のデジタル資産はグローバルネットワーク上で自律的に運営されており、基本的に規制されていません。取引者は、ピアツーピアネットワークと暗号技術に依拠してその完全性を維持する、デジタル化された分散型かつ部分的に匿名のシステムに信頼を置いています。

-デジタル資産の取引は不可逆的である場合があり、そのため、詐欺または誤った取引による損失は回復できない可能性があります。

-一部のデジタル資産取引は、お客様が取引を開始した日時ではなく、公開台帳に記録された時点で実行されたものとみなされます。

-デジタル資産の価値は、市場参加者が法定通貨またはデジタル資産をデジタル資産に交換し続ける意思に由来する場合があり、特定のデジタル資産の市場が消滅した場合、当該デジタル資産の価値が永久的かつ完全に喪失する可能性があります。

-デジタル資産の性質上、詐欺やサイバー攻撃のリスクが増大する可能性があり、プラットフォームが経験する技術的問題により、お客様のデジタル資産にアクセスまたは使用できなくなる場合があります。

-デジタル資産取引は、非合理的な（または合理的な）バブルや信頼の喪失の影響を受けやすく、供給に対する需要の崩壊につながる可能性があります。例えば、ソフトウェア開発者やその他の関係者による予期せぬ変更、政府による取り締まり、優れた競合暗号通貨の出現、デフレまたはインフレの悪循環により、ビットコインへの信頼が崩壊する可能性があります。システムの匿名性が損なわれた場合、資金が紛失または盗難された場合、あるいはハッカーや政府が取引の決済を阻止できる場合にも、信頼が崩壊する可能性があります。

-お客様の管轄区域で適用される法律、規制または規則に基づき、お客様のアカウント内の預金は預金とみなされない場合があります。

-お客様のアカウント内のデジタル資産は、適用される預金保険制度の保護対象ではありません。
