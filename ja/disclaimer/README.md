管轄当局が関連する調査証明書に基づき、本ウェブサイトに対して特定のユーザーに関する調査への協力を要請した場合、またはユーザーアカウントの閉鎖、凍結、もしくは送金等の措置を講じる必要がある場合、本ウェブサイトは管轄当局の要請に従い、当該アカウントユーザーに関するデータを提供し、または管轄当局の要請に基づき関連する措置を実施いたします。本ウェブサイトは、上記の行為に起因するユーザーのプライバシー開示、アカウント操作の失敗、および損失について、いかなる形態の責任も負いません。
