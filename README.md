# Koriex Help Center

Koriex 交易平台帮助中心文档仓库。

本项目使用 [GitBook](https://www.gitbook.com/) 构建，支持以下语言：

- 简体中文
- English
- 日本語
- 한국어

## 本地预览

```bash
npx gitbook-cli install
npx gitbook-cli serve
```

## 目录结构

每种语言包含以下模块：

- **关于 Koriex** — 平台介绍
- **风控规则** — 风险控制相关规则
- **免责声明** — 法律免责条款
- **服务条款** — 用户服务协议
- **隐私政策** — 数据隐私保护政策
